# PROJET-ANNUEL-IABD
Problématique de classification d’images : distinguer automatiquement avion / voiture / bateau à partir d’une photo. Constituer un dataset d’images représentatif, implémenter et comparer plusieurs modèles vus en cours, puis exposer les modèles via une API qui permet à l’utilisateur d’envoyer une image et d’obtenir la prédiction
